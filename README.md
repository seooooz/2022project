# 4D; For Developer
> 2022.09.08 - 2022.10.26

## Description
> 예비 개발자들을 위한 커뮤니티 사이트 학원, 관려 시험 등 정보를 찾는 것에 불편을 느껴 만들게 되었습니다. 

## 기능 설명
> 학원, 관련 시험 정보, 공모전 등 정보 제공
> CRUD 게시판 제공

## 개발 환경

OS X & 리눅스:

```sh
npm install my-crazy-module --save
```

윈도우:

```sh
edit autoexec.bat
```
